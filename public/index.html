Hitch a Ride

Wedding Carpooling

Hitch a Ride is a ride board I built for my wedding to help guests get to our remote venue. It uses React, routing, Bootstrap, SASS, and Google Authentication to help users fill and find open seats in rides to and from our wedding. This was my front-end capstone at Nashville Software School.

Visit website GitHub repository

MovieMinder

Helping you see films in theaters

Seeing movies in theaters is a special experience, and seeing them on opening weekend is a great way to show support for your favorite creators. Keeping up with all the movies you want to see and their release dates, or even just remembering to see them before they ride off into the sunset can be daunting. MovieMinder was built to solve this problem, allowing users to track movies they want to watch before, during, and even after they're at the local theater. MovieMinder is built with a React.js and Boostrap front-end, an ASP.NET Core back-end, and a SQL database.

GitHub repository

Chatty

A themed chat app

Chatty was a group project I was a part of where we designed a web-app that commited messages to a database and did some basic permissions checking to make sure you couldn't delete other people's messages

My contribution to the project was the theme selection and creation. There is a light theme, a dark theme, and an option to create a custom theme. The custom theme option lets the user choose a primary and secondary color using the built-in color picker from their OS.
